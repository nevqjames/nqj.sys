HOLOLIVE
SUPER EXPO

Layout & UI Design for the premier global VTuber event.

Role

UI Designer / Frontend Integrator

Client

COVER Corp (Japan)

Timeline

4 Months (Pre-event)

Tools

Figma, SCSS, JS

Project Overview

The Hololive Super Expo is a massive annual event celebrating the VTuber culture. Working within a cross-functional international team, I was tasked with architecting the visual layout and interaction design of the primary landing page. The goal was to translate the vibrant, high-energy "Idol" aesthetic into a functional, high-performance web experience that could handle millions of simultaneous visitors.

Design Process (Figma)

My workflow began in Figma, where I developed a modular design system. I focused on creating a "Digital Festival" atmosphere using neon accents and layered glassmorphism. I was responsible for:

// FIGMA WORKSPACE: WIREFRAMES & PROTOTYPES

[ FIGMA LAYOUT VISUALIZATION ]

  • Hierarchical Grid Systems for schedule-heavy data.
  • Responsive breakpoints for seamless mobile browsing.
  • Interactive component states (Hovers, Taps, Loading).
  • Asset optimization for high-density displays.

Technical Implementation

[ CRITICAL CHALLENGE ]

Designing for a global audience meant the UI had to be incredibly lightweight despite the heavy visual effects. I bridged the gap between design and dev by writing the initial SCSS/JS components.

I translated the Figma designs into high-performance code, focusing on the "Look and Feel" of the scrolling experience. Using CSS Grid and Custom Properties, I ensured the theme could be updated instantly across the site for different talent group reveals.

// CODE SNIPPET: RESPONSIVE GRID ARCHITECTURE 
.expo-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
  gap: var(--spacing-xl);
  perspective: 1000px;
}

.talent-card {
  transition: transform var(--ease-expo);
  backdrop-filter: blur(10px);
}

Key Results

1.2M+ Unique Hits (Day 1)
98 Lighthouse Perf Score

The project was successfully launched globally, receiving praise for its intuitive navigation of the complex event schedule. My role as the design-to-code bridge reduced the production time by 3 weeks, as the engineering team could directly utilize the CSS components I developed.
